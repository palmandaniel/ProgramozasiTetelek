# Programozási tételek

## Elõkészítés

- tíz elemû egész számot tartalmazó tömb létrehozása
- Tömb elemei:
    - 12, 7, 2, 4, 45, 99, 1, 8, 95, 50
- A tömb az összes metódusban látható legyen
- TombKiiras() metódus létrehozása
    -nem ad vissza értéket
    -nincs paramétere
    -a tömb elemeit vesszõvel elválasztva a console-ra írja

## Összegzés
- összegzés metódus létrehozása
- összegzés tétele a tömbön
- eredmény kiírása a console-ra

## Megszámolás
- Megszamolas() metódus létrehozása
- Mennyi 3-al osztható elem van a tömbben
- eredmény kiírása a console-ra

## Eldöntés
- Eldontes() metódus lértehozása
- Van-e a tömbben 69? 
- Kiírni: "van" / "nincs"

## Kiválasztás
- Kivalasztas() metódus 
- Hányadik az 50?
- Kiírni a helyét a tömbben

## Keresés
- Kereses() metódus
- van-e benne 50, és ha van, hányadik?

## Max
- Max() metódus megkeresése
- kiírni a tömb legnagyobb elemét
